# Chinese Belt and Road Initiative Port Infrastructure in East Africa

This project examines infrastructural and geostrategic characteristics of three BRI-associated ports in East Africa using QGIS.

## Research Question

What do the spatial characteristics of selected BRI-associated ports reveal about China's maritime infrastructure presence in East Africa?

## Software

- QGIS
- Microsoft Excel
- Microsoft Word
- Python
